# Store Manager - Sistema de Gestión Comercial

**Store Manager** es una herramienta gratuita y potente diseñada para simplificar la administración de tu negocio. Olvida los registros en papel y profesionaliza tu mostrador hoy mismo.

---

## Funciones Destacadas

- **Ventas Rápidas:** Compatible con lectores de códigos de barras.
- **Stock Bajo Control:** Inventario en tiempo real con alertas.
- **Reportes en PDF:** Imprime tickets y cierres de caja profesionales.
- **Etiquetas:** Genera tus propios códigos de barras listos para imprimir.
- **Seguridad:** Copias de respaldo de toda tu información en un clic.

---

## Descarga e Instalación

Para empezar a usar el sistema en tu negocio, sigue estos pasos:

1. Ve a la sección de **[Última Versión (Releases)](https://github.com/gocendra/StoreManager/releases/tag/V1.7)**.
2. Descarga el archivo `StoreManager.exe`.
3. Ejecútalo en tu PC con **Windows 10 u 11**.
4. ¡Comienza a vender! No requiere configuraciones complejas ni bases de datos externas.

---

## 🛡️ Seguridad y Privacidad
Toda la información se guarda de forma **local** en tu computadora. El sistema no requiere internet para funcionar y tus datos de ventas no salen de tu local.
